# README
booksテーブルと、そのCRUDを行うAPIの作成。

●全件取得
　GET /api/v1/books
●著者名検索
　GET /api/v1/books/著者名
●新規登録
　POST /api/v1/books/new
●データ更新
　PUT /api/v1/books/update
●データ削除
　DELETE /api/v1/books/delete
